# fusion

### Install Guidelines

```
git clone https://github.com/HenryLHH/fusion

# install fusion package in the virtual env
cd fusion
conda create -n fusion
conda activate fusion
pip install -e .[all]

# train the agents
bash run_fusion.sh
```
